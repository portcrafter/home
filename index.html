Frequently Asked Questions

Find answers to common questions about PortCrafter

Do I need coding skills to use PortCrafter?

No coding skills are required! Our intuitive editor lets you build a professional portfolio through a simple point-and-click interface. However, if you do know how to code, we offer advanced customization options through CSS and JavaScript.

Can I use my own domain name?

Yes! All paid plans allow you to connect your own custom domain name (e.g., yourname.com). Our free plan includes a portcrafter.io subdomain (e.g., yourname.portcrafter.io).

How do I transfer my existing portfolio?

We provide easy import tools for portfolios hosted on other platforms. For custom sites, our support team can help you migrate your content. Most users find they can recreate their portfolio in PortCrafter in just a few hours.

What happens if I cancel my subscription?

If you cancel, your portfolio will revert to the free plan (with applicable limitations). We never delete your work - you can export your content at any time or upgrade again later.

How does the GitHub Pages integration work?

PortCrafter automatically creates a GitHub repository for your portfolio, sets up GitHub Pages, and handles all deployments. You get all the benefits of GitHub Pages (free hosting, custom domain, etc.) without any of the setup complexity.
